--- title: layout: post --- '2010-07-23 07:58:41','Sound in Linux is a disaster. There are many different software packages (OSS, ALSA, PulseAudio, ESD, GStreamer etc.) and it is very hard for the beginner to figure out what he needs. It took me a while but I eventually worked it out. See this opinionated post on the history of sound in Linux by the creator of OSS. I will give a short recap here.\r\n\r\nOnce upon a time, long, long ago, there was OSS, the Open Sound System. It worked on many OS\'s including Linux. But then some Linux users created another sound API called ALSA, the Advanced Linux Sound Architecture and deprecated OSS. The creator of OSS joined a company called 4Front Technology to commercially develop OSS. Recently (2007 or 2008 I think), OSS was once again open sourced.\r\n\r\nLong story short, ALSA sucks and OSS rocks. I had many problems with ALSA, for instance it allowing only one audio source to play at one time. So I removed it altogether and installed the new OSS .deb from 4Front / opensound.com. This instantly worked like a charm!\r\n\r\nFollow the excellent Ubuntu community guide to install OSS on Ubuntu. If you have been following along with my previous post for a minimal install, then you don\'t have to remove PulseAudio, ALSA and GStreamer. Just do the linux-base-sound step and install the .deb from 4Front.\r\n\r\nWhen you\'re done, test sound with osstest or open a graphical mixer with ossxmix. To setup applications to use OSS, see the 4Front wiki.\r\n

MPD

\r\nNow rejoice! Or install MPD with\r\n\r\nsudo aptitude install mpd.\r\n\r\nEdit /etc/mpd.conf after the install to point to your music directory, remove the ALSA output section and uncomment the OSS output section. Now tell it to do an initial scan of your music directory with\r\n\r\nsudo /etc/init.d/mpd stop; sudo /etc/init.d/mpd start-create-db.\r\n\r\nIt will probably also install mpc, a simple command line client for MPD. Try\r\n\r\nmpc search artist astley | mpc add\r\nmpc play .\r\n\r\nNormally PMS is a bad thing, but Practical Music Search is not. It is a really cool and simple client for MPD. It is not available as a Ubuntu package as far as I know, so download and install from source. It should be a simple\r\n\r\nconfigure && sudo make install.\r\n\r\nNote that you need glib2 and ncurses. So do\r\n\r\nsudo aptitude install glib2-dev ncurses-dev\r\n\r\nor something similar.
