--- title: layout: post --- '2010-07-23 07:52:55','First, install the gemcutter and jeweler gems:\r\n
$ sudo gem install gemcutter jeweler
\r\nNow signup at gemcutter.org\r\n\r\nNext, create a jeweler project:\r\n
$ jeweler mygem\r\n    create  .gitignore\r\n    create  Rakefile\r\n    create  LICENSE\r\n    create  README.rdoc\r\n    create  .document\r\n    create  lib\r\n    create  lib/mygem.rb\r\n    create  test\r\n    create  test/helper.rb\r\n    create  test/test_mygem.rb
\r\nThis will generate a Rakefile and test and lib directories. The Rakefile includes gemcutter support. Run rake -T to see all rake tasks.\r\n\r\nCustomize the generated Rakefile and add your tests and implementation. When you are ready to push to gemcutter, set a version and push:\r\n
$ rake version:bump:minor\r\n$ rake gemcutter:release
\r\nI think the release step will prompt you for your gemcutter credentials.\r\n\r\nThat\'s it!','Gemcutter + Jeweler gem publishing mini-howto','','publish','open','open','','gemcutter-jeweler-gem-publishing-mini-howto','','','2010-07-23 07:52:55','2010-07-23 07:52:55','',0,'http://blog.ronaldevers.nl/?p=15',0,'post','',0);
